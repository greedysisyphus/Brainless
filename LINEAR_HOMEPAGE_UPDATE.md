# Linear 風格首頁更新說明

## ✅ 已完成的工作

### 1. 創建 Linear 風格首頁 (`src/pages/HomepageLinear.jsx`)

#### Hero 區域
- **大標題設計**：模仿 Linear 的 "Linear is a purpose-built tool" 風格
- **副標題**：簡潔的描述文字
- **CTA 按鈕**：主要和次要按鈕，帶懸停動畫
- **滾動視差效果**：使用 Framer Motion 的 `useScroll` 和 `useTransform`
- **動態背景**：漸變光暈效果，循環動畫
- **滾動提示**：底部滾動指示器動畫

#### 功能展示區域
- **網格布局**：響應式 3 列卡片網格
- **滾動觸發動畫**：卡片進入視窗時淡入和上移
- **懸停效果**：卡片懸停時上移和縮放
- **漸變背景**：每個卡片有不同的漸變色

### 2. 優化 Header 和 Navigation

#### HeaderLinear 改進
- **首頁模式**：在首頁時，Header 背景變為透明
- **可點擊 Logo**：點擊 Logo 返回首頁
- **條件顯示**：首頁時隱藏時鐘和分隔線

#### NavigationLinear 改進
- **首頁隱藏**：在首頁時完全隱藏 Navigation
- **平滑過渡**：使用 CSS transition 實現平滑顯示/隱藏

#### LayoutLinear 改進
- **首頁布局**：首頁時移除內邊距和容器限制
- **全屏顯示**：讓首頁可以佔滿整個視窗

### 3. 路由更新

- 在 Linear 主題下，根路徑 (`/`) 顯示 `HomepageLinear`
- 在經典主題下，根路徑繼續重定向到 `/sandwich`
- 使用條件渲染：`theme === 'linear' ? <HomepageLinear /> : <Navigate to="/sandwich" />`

## 🎨 設計特色

### 視覺效果
- **極簡設計**：大量留白，清晰的層次
- **平滑動畫**：使用 Framer Motion 實現流暢過渡
- **滾動視差**：Hero 區域隨滾動產生視差效果
- **微交互**：懸停、點擊都有細膩的反饋

### 動畫效果
- **進入動畫**：元素淡入和上移
- **懸停動畫**：按鈕和卡片有縮放效果
- **滾動動畫**：基於滾動位置的視差效果
- **循環動畫**：背景光暈和滾動提示器

## 📝 使用方式

1. **切換到 Linear 主題**：點擊 Header 右上角的主題切換按鈕
2. **訪問首頁**：在 Linear 主題下，訪問根路徑 (`/`) 會顯示新的首頁
3. **瀏覽功能**：滾動查看功能卡片，點擊卡片進入對應功能頁面

## 🚀 技術實現

### 使用的庫
- **Framer Motion**：所有動畫效果
  - `useScroll`：追蹤滾動位置
  - `useTransform`：根據滾動位置變換值
  - `useInView`：檢測元素是否在視窗內
  - `motion`：動畫組件
  - `AnimatePresence`：頁面轉場

### 動畫技術
- **視差滾動**：Hero 區域內容隨滾動移動
- **滾動觸發**：卡片在進入視窗時觸發動畫
- **漸變動畫**：背景光暈的循環動畫
- **懸停反饋**：按鈕和卡片的交互動畫
