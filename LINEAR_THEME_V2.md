# Linear 風格主題 V2 - 極簡設計實現

## 🎯 目標
實現與 Linear 官網非常相似的極簡設計風格。

## ✅ 已完成的工作

### 1. 創建專用的 Linear 風格組件

#### HeaderLinear (`src/components/layout/HeaderLinear.jsx`)
- **極簡設計**：移除多餘裝飾
- **水平布局**：Logo、標題、時鐘在同一行
- **高度固定**：`h-16` (64px)
- **背景**：`bg-[#0a0a0a]/80` 帶毛玻璃效果
- **邊框**：極細邊框 `border-white/[0.06]`
- **主題切換按鈕**：簡化的圖標按鈕

#### NavigationLinear (`src/components/layout/NavigationLinear.jsx`)
- **水平導航**：所有項目在同一行
- **動態指示器**：底部滑動的活動指示條（2px 高度）
- **極簡樣式**：
  - 活動狀態：`text-white`
  - 非活動狀態：`text-white/60`
  - 懸停：`hover:text-white/90 hover:bg-white/[0.06]`
- **圖標 + 文字**：每個項目都有圖標和標籤

#### LayoutLinear (`src/components/layout/LayoutLinear.jsx`)
- **背景顏色**：`#0a0a0a` (更柔和的深灰色)
- **極淡網格**：透明度 `0.015`，網格大小 `32px`
- **移除多餘裝飾**：去掉漸變光暈和複雜動畫
- **簡化頁面轉場**：更快的動畫（0.3s），更小的位移（10px）

### 2. 設計特色

#### 顏色方案
- **背景**：`#0a0a0a` - Linear 風格的深灰背景
- **邊框**：`rgba(255, 255, 255, 0.06)` - 極細的邊框
- **文字**：
  - 主要：`#ffffff`
  - 次要：`rgba(255, 255, 255, 0.6)`

#### 間距和布局
- **Header 高度**：64px
- **Navigation 高度**：48px
- **內容間距**：`py-12` (48px 上下)
- **水平間距**：`px-6` (24px 左右)
- **最大寬度**：`max-w-7xl`

#### 動畫和交互
- **頁面轉場**：0.3s，easeOutCubic
- **導航指示器**：彈簧動畫（spring），流暢滑動
- **懸停效果**：200ms 快速過渡

### 3. 與 Linear 官網的相似之處

✅ **極簡的 Header**：簡潔的 Logo 和標題，水平布局
✅ **水平導航**：所有項目在一行，帶動態指示器
✅ **深色背景**：接近 Linear 的 `#0a0a0a`
✅ **極細邊框**：幾乎看不見的邊框分隔
✅ **大量留白**：內容區域有充足的間距
✅ **微妙的交互**：快速、流暢的懸停效果
✅ **簡約的動畫**：不張揚的轉場效果

## 📝 使用方法

1. **切換到 Linear 主題**
   - 點擊 Header 右上角的主題切換按鈕
   - 選擇 Linear 風格

2. **查看效果**
   - Header 和 Navigation 會自動切換到 Linear 風格
   - 頁面內容會使用 Linear 的背景和樣式
